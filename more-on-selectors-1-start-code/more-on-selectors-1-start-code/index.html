This is the intro section.

Sự khác biệt giữa class và ID.

  • Class co thể sử dụng lại được nhiều lần(tái sử dụng)
  • Class chỉ sử dụng cho mục đích định dạng
  • Class được sử dụng nhiều nhất trong các kiểu selector
  • ID không thể sử dụng lại nhiều lần được
  • Không nên sử dụng ID trừ những trường hợp bất đắt dĩ.
  • Thường sử dụng để làm link tới các phần của trangq
  • Ta không nên dùng !important, vì cái này cũng rất nguy hiểm
